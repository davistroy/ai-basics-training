# AI Basics Training

A comprehensive training program for AI-assisted development, workflow engineering, and automation architecture.

## Program Structure

### Core Curriculum (Blocks 1-3)

The foundational training is organized into three progressive blocks:

| Block | Title | Duration | Focus |
|-------|-------|----------|-------|
| [Block 1](block-1.md) | AI-Assisted Development Fundamentals | 6 weeks | Prompt engineering, AI tool usage, structured outputs |
| [Block 2](block-2.md) | AI Workflow Engineering | 6 weeks | Workflow design, multi-step automation, testing |
| [Block 3](block-3.md) | AI Automation Architecture | 6 weeks | Orchestration patterns, system design, production deployment |

### Advanced Modules

Optional deep-dive modules for specialized topics (requires Block 3 completion):

| Module | Title | Duration | Focus |
|--------|-------|----------|-------|
| [Module 1](advanced-modules/module-1-orchestration-patterns.md) | Advanced Orchestration Patterns | 2 weeks | Parallel & Team-Based orchestration |
| [Module 2](advanced-modules/module-2-chaos-testing.md) | Chaos Testing & Reliability | 2 weeks | Failure injection, circuit breakers, self-healing |
| [Module 3](advanced-modules/module-3-json-schema.md) | JSON Schema & Data Validation | 2 weeks | Schema design, validation pipelines, error handling |
| [Module 4](advanced-modules/module-4-visual-documentation.md) | Visual Documentation (Mermaid) | 2 weeks | Flowcharts, sequence diagrams, architecture diagrams |
| [Module 5](advanced-modules/module-5-github-devops.md) | Advanced GitHub & DevOps | 2 weeks | CI/CD pipelines, GitHub Actions, deployment strategies |

## Prerequisites

- Basic programming experience
- Familiarity with command-line tools
- Access to AI coding assistants (Claude, GitHub Copilot, etc.)

## Learning Path

```
Block 1 (6 weeks)
    ↓
Block 2 (6 weeks)
    ↓
Block 3 (6 weeks)
    ↓
Advanced Modules (optional, 2 weeks each)
```

1. Begin with [Block 1](block-1.md) to establish foundational AI collaboration skills
