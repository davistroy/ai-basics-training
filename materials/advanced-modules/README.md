# Advanced Modules

Ten optional post-Block 3 modules for specialized skill development. Each module is **2 weeks** with 45-minute live sessions and 60-75 minutes of homework per week.

---

## Prerequisites

All advanced modules require completion of the core curriculum:
- **Minimum**: Block 2 Certification (AI Workflow Engineer) for Modules 3, 4, 7
- **Recommended**: Block 3 Certification (AI Automation Architect) for Modules 1, 2, 5, 6, 8, 9, 10

## Module Details

### Module 1: Advanced Orchestration Patterns
**Target Audience:** Block 3 graduates implementing enterprise-scale multi-agent automation

Master sophisticated multi-agent coordination including Parallel orchestration for concurrent tasks, Team-Based patterns for collaborative workflows, hybrid orchestration strategies, and fault-tolerant multi-agent systems.

**Capstone:** Multi-Pattern Orchestration System

---

### Module 2: Chaos Testing & Reliability Engineering
**Target Audience:** Block 3 graduates building production-grade AI agent systems

Learn chaos engineering principles for AI systems, systematic failure injection testing, advanced reliability patterns (circuit breakers, bulkheads), and self-healing agent architectures.

**Capstone:** Reliability Engineering Report

---

### Module 3: JSON Schema & Data Validation
**Target Audience:** Block 2+ graduates implementing robust data validation in workflows

Master JSON Schema specification, create schemas for AI workflow data structures, implement validation in automation workflows, and build reusable schema libraries.

**Capstone:** Validated Workflow System

---

### Module 4: Visual Documentation with Mermaid
**Target Audience:** Any Block 1+ graduate creating professional visual documentation

Learn Mermaid diagram syntax for flowcharts, sequence diagrams, and architecture diagrams. Integrate diagrams into GitHub documentation for workflows and agent architectures.

**Capstone:** Visual Documentation Suite

---

### Module 5: Advanced GitHub & DevOps for AI Systems
**Target Audience:** Block 3 graduates implementing professional DevOps practices

Design GitHub Actions workflows for AI automation, version control best practices for prompts and configurations, CI/CD pipelines for AI validation and deployment, and automated testing for AI systems.

**Capstone:** Production-Ready AI DevOps System

---

### Module 6: Enterprise AI Architecture
**Target Audience:** Block 3 graduates advising on AI platform selection and deployment

Evaluate major cloud AI platforms (AWS Bedrock, Google Vertex AI, Azure OpenAI), design hybrid and on-premises architectures, create cost models, and build platform selection decision frameworks.

**Capstone:** Enterprise AI Architecture Proposal

---

### Module 7: Enterprise AI Governance
**Target Audience:** Consulting professionals advising on AI governance and compliance

Master AI risk frameworks (NIST AI RMF, ISO 42001), implement responsible AI principles, design governance lifecycles, navigate regulatory compliance (EU AI Act), and conduct governance maturity assessments.

**Capstone:** Enterprise AI Governance Framework

---

### Module 8: RAG & Knowledge Systems
**Target Audience:** Block 3 graduates building knowledge-enabled AI agents

Understand RAG architecture patterns, design chunking and embedding strategies, implement retrieval patterns (semantic, hybrid, re-ranking), and integrate knowledge systems with agent workflows.

**Capstone:** Knowledge-Enabled Agent System

---

### Module 9: AI Security Fundamentals
**Target Audience:** Block 3 graduates designing secure agent architectures

Understand AI/LLM threat landscape, defend against prompt injection attacks, apply agent security patterns (least privilege, sandboxing), implement data security controls, and conduct security assessments.

**Capstone:** AI Security Assessment Report

---

### Module 10: Data Architecture for AI Systems
**Target Audience:** Block 3 graduates designing data structures for AI agent consumption

Design information architectures for AI navigation, apply semantic structuring patterns, select appropriate data formats, implement metadata strategies, and build data transformation pipelines.

**Capstone:** AI-Ready Data Architecture

---

## Recommended Learning Paths

### Security-Focused Path
Module 9 (Security) → Module 7 (Governance) → Module 2 (Reliability)

### Enterprise Architecture Path
Module 6 (Enterprise Architecture) → Module 10 (Data Architecture) → Module 8 (RAG)

### DevOps Path
Module 5 (GitHub/DevOps) → Module 2 (Chaos Testing) → Module 1 (Orchestration)

### Documentation & Standards Path
Module 4 (Mermaid) → Module 3 (JSON Schema) → Module 7 (Governance)

## Time Commitment

- **Standard modules**: 45 min live + 75 min homework/week (4 hours total)
- **Reduced homework modules** (3, 4): 45 min live + 60 min homework/week (3.5 hours total)

Each module runs independently, allowing flexible scheduling based on team needs and priorities.
